## Возможности
- Просмотр текущей погоды
- Прогноз на 7 дней
- Поиск городов
- Определение геолокации

## Технологии
- Swift
- UIKit
- SnapKit
- CoreLocation

## Архитектура
- **MVVM** — разделение бизнес-логики и отображения
- **Coordinator** — управление навигацией между экранами
- **Bindable** — связывание ViewModel и View
